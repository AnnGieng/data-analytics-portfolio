This repository showcases my data analysis and visualization projects using **Microsoft Excel** and **Power BI**. Each project focuses on transforming raw data into meaningful insights to support decision-making.

## 🔹 1. Bike Sales Dashboard (Excel)

### 📌 Overview

This project analyzes customer data to understand the factors influencing bike purchases. The dashboard provides interactive filtering based on demographics and customer behavior.

### 🛠 Tools Used

* Microsoft Excel (Pivot Tables, Charts, Slicers)

### 📈 Key Insights

* Customers in the **middle-age bracket** show the highest likelihood of purchasing bikes
* **Higher income individuals** are more likely to purchase bikes, especially males
* Shorter commute distances (0–5 miles) are associated with higher bike purchases
* Occupation and region significantly influence purchasing patterns

### 🎛 Features

* Interactive filters for:

  * Marital Status
  * Occupation
  * Region
* Dynamic charts:

  * Average Income per Purchase
  * Customer Age Bracket
  * Customer Commute Distance

## 🔹 2. Data Professional Survey Breakdown (Power BI)

### 📌 Overview

This dashboard analyzes survey data from data professionals to uncover trends in salaries, job roles, programming preferences, and job satisfaction.

### 🛠 Tools Used

* Power BI (Data Modeling, DAX, Interactive Visualizations)

### 📈 Key Insights

* **Data Scientists** have the highest average salaries compared to other roles
* **Python** is the most preferred programming language among data professionals
* A significant portion of respondents find it **neither easy nor difficult** to break into data careers
* Work-life balance satisfaction is relatively moderate, while salary satisfaction is lower
* Respondents are distributed globally, with notable representation from **Canada, India, the UK, and the US**

### 🎛 Features

* Interactive visualizations:

  * Salary distribution by job title
  * Country distribution (Treemap)
  * Programming language preferences
  * Career difficulty perception
* KPI indicators:

  * Total survey participants
  * Average age
  * Work-life balance rating
  * Salary satisfaction rating
